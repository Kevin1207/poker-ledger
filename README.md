# 学习记账本

一个简洁的微信小程序，用于记录学习的输赢情况，帮助你更好地管理和分析你的学习战况。

## 功能特性

✅ **快速记录**
- 支持输赢金额的快速录入
- 可添加备注信息（场次类型、对手等）
- 自定义记录日期

✅ **实时统计**
- 本月战况实时显示
- 按月统计：查看每月的输赢情况、胜率分析
- 按年统计：查看全年的战况总结

✅ **数据可视化**
- 清晰的卡片式设计
- 红绿配色直观展示盈亏
- 胜率计算自动展示

✅ **本地存储**
- 使用微信本地存储，数据安全可靠
- 无需联网即可使用

## 快速开始

1. **安装微信开发者工具**
   - 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

2. **导入项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择本项目目录 `/Users/kevin1207/poker-ledger`
   - AppID 可以选择"测试号"或使用自己的 AppID

3. **开始使用**
   - 在模拟器或真机中预览
   - 点击"添加记录"开始记录你的学习战况
   - 在"统计"标签查看详细的月度/年度统计

## 使用说明

### 添加记录
1. 点击首页的"添加记录"按钮
2. 选择"赢"或"输"
3. 输入金额（只输入数字，符号会自动添加）
4. 可选：添加备注（如"线下局"、"好友局"等）
5. 选择日期（默认为当天）
6. 点击"保存记录"

### 查看统计
- **首页**：查看本月总战况和最近20条记录
- **统计页面**：
  - 按月统计：查看每个月的详细数据
  - 按年统计：查看每年的总体情况
  - 包含总金额、次数、胜率等指标

## 数据说明

- 数据存储在微信本地，不会上传到服务器
- 卸载小程序或清除缓存会导致数据丢失
- 建议定期截图保存重要数据

## 技术栈

- 微信小程序原生开发
- JavaScript ES6+
- WXML + WXSS
- 本地存储（wx.setStorageSync / wx.getStorageSync）
