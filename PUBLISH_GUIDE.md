# 📱 小程序发布指南

恭喜你完成开发！现在来发布你的学习记账本小程序吧！

---

## 🚀 发布流程总览

```
1. 上传代码到微信服务器
   ↓
2. 在小程序后台提交审核
   ↓
3. 等待审核（1-7天）
   ↓
4. 审核通过后发布上线
   ↓
5. 用户可以搜索和使用
```

---

## 📤 第一步：上传代码

### 在微信开发者工具中：

1. **点击工具栏的"上传"按钮**
   - 位置：顶部工具栏，"预览"按钮旁边
   - 图标：一个向上的箭头

2. **填写版本信息**
   - **版本号**：`1.0.0`（首次发布建议用这个）
   - **项目备注**：`首次发布，包含记录、统计、云开发功能`

3. **点击"上传"**
   - 等待上传完成
   - 成功后会显示"上传成功"

---

## 🖥️ 第二步：登录小程序后台

1. **打开微信公众平台**
   - 网址：https://mp.weixin.qq.com/
   - 使用管理员微信扫码登录

2. **进入"版本管理"**
   - 左侧菜单：管理 → 版本管理

3. **找到"开发版本"**
   - 应该能看到刚才上传的版本（1.0.0）

---

## ✅ 第三步：提交审核

### 1. 完善小程序信息（首次发布必须）

在提交审核前，需要先完善基本信息：

**进入"设置" → "基本设置"**

#### A. 小程序信息
- **小程序名称**：学习记账本（或你喜欢的名字）
- **小程序头像**：上传一个图标（建议 200x200 像素）
- **小程序简介**：学习记录管理工具

#### B. 服务类目（重要！）

**路径**：设置 → 基本设置 → 服务类目

**推荐选择**：
- **工具** → **记账软件**
- 或 **工具** → **效率**

**⚠️ 避免选择**：
- 任何与"赌博"、"博彩"相关的类目
- 这会导致审核被拒

#### C. 用户隐私保护指引（必须！）

**路径**：设置 → 基本设置 → 用户隐私保护指引

**填写内容**：

**隐私政策链接**：
```
pages/privacy/privacy
```

**收集信息类型**：
- ✅ 位置信息：否
- ✅ 个人身份信息：否
- ✅ 设备信息：否
- ✅ 用户输入内容：是（学习记录数据）

**收集目的**：
```
用于在小程序内展示和统计用户的学习记录
```

---

### 2. 提交审核

**在"版本管理" → "开发版本"中**：

1. **点击"提交审核"按钮**

2. **配置功能页面**

   需要添加主要功能页面的路径：

   | 功能名称 | 页面路径 | 页面标题 |
   |---------|---------|---------|
   | 记录列表 | pages/index/index | 学习记账本 |
   | 添加记录 | pages/add/add | 添加记录 |
   | 统计分析 | pages/statistics/statistics | 战况统计 |

3. **填写审核信息**

   **版本描述**：
   ```
   学习记账工具，支持记录每次游戏的输赢情况，
   自动统计月度、年度战况和胜率。数据保存在云端，
   支持跨设备同步。
   ```

   **测试账号**：
   - 如果小程序不需要登录，选择"无需测试账号"
   - 我们的小程序不需要特殊权限，选择"无需测试账号"

4. **补充说明（可选但推荐）**

   ```
   1. 本小程序为个人记账工具，仅记录学习游戏结果
   2. 不涉及任何赌博、金钱交易功能
   3. 所有数据仅供用户个人查看和统计
   4. 使用微信云开发存储用户数据
   ```

5. **点击"提交审核"**

---

## ⏳ 第四步：等待审核

### 审核时长
- **一般**：1-3 个工作日
- **可能**：3-7 个工作日
- **特殊情况**：可能需要补充资料

### 审核结果查看
- 路径：管理 → 版本管理 → 审核版本
- 会收到微信通知（模板消息）

---

## 🎉 第五步：发布上线

### 审核通过后：

1. **进入"版本管理"**
2. **找到"审核通过"的版本**
3. **点击"发布"按钮**
4. **确认发布**

**恭喜！** 你的小程序已上线！🎊

## ⚠️ 审核注意事项

### 1. 名称和描述避免敏感词

**避免使用**：
- ❌ 赌博、博彩、下注
- ❌ 真金、现金、提现
- ❌ 赚钱、盈利、收益

**推荐使用**：
- ✅ 记录、统计、分析
- ✅ 游戏、娱乐、练习
- ✅ 管理、工具、助手

### 2. 功能说明要清晰

**强调**：
- 这是一个**记录工具**，不涉及金钱交易
- 用于**个人统计分析**，不涉及赌博
- 只是**数据管理**，不提供任何游戏功能

### 3. 隐私政策必须有

- 已经创建了 `pages/privacy/privacy` 页面
- 在审核时务必填写隐私政策链接

### 4. 类目选择要准确

- 不要选择"游戏"类目（需要游戏版号）
- 不要选择"金融"类目（需要资质）
- 推荐：工具 → 记账软件 / 效率

---

## 🚫 常见审核拒绝原因

### 1. 涉及赌博内容
**解决**：
- 强调这是"记录工具"，不涉及赌博
- 修改描述，避免敏感词
- 补充说明仅用于个人统计

### 2. 缺少隐私政策
**解决**：
- 已创建隐私政策页面
- 在审核时填写隐私政策链接

### 3. 功能页面无法访问
**解决**：
- 确保云开发环境已开通
- 确保数据库集合已创建
- 提供测试账号（如需要）

### 4. 类目不符
**解决**：
- 修改服务类目为"工具 → 记账软件"
- 重新提交审核

---

## 📞 审核被拒后怎么办？

### 1. 查看拒绝原因
- 在"版本管理"中查看详细原因
- 微信也会发送通知

### 2. 根据原因修改
- 按照审核意见修改代码或说明
- 重新上传版本

### 3. 重新提交
- 修改后再次点击"提交审核"
- 在"补充说明"中说明已修改内容
